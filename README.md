# AI Workspace

这是一个用于沉淀我在 **AI 开发与实际使用** 过程中的资料仓库，主要包含：

- Agent 协作规范
- 可复用 Skills
- 评审/实验相关模板与说明

## 仓库定位

本仓库用于持续积累以下内容：

- AI 工程化实践（流程、规范、任务拆解）
- Agent 技能定义与复用
- 开发中的经验与复盘

## 当前目录结构

```text
.
├── AGENTS.md
├── README.md
└── skills/
    └── candidate-code-reviewer/
        └── SKILL.md
```

## 目录说明

- `AGENTS.md`

  - 仓库级协作与执行规范（语言、工作流、任务管理等）。

- `skills/`
  - 存放可复用技能定义。
  - 当前已包含：`candidate-code-reviewer`（候选人代码评审技能）。

## 使用方式

1. 阅读 `AGENTS.md`，了解仓库内统一协作规则。
2. 在 `skills/` 下按目录新增或维护技能，每个技能使用独立 `SKILL.md` 说明。
3. 新增任务时，按约定在 `tasks/` 目录创建 `todo-*.md` 跟踪计划与复盘。

## 维护原则

- 简单优先，尽量最小化改动范围。
- 先计划再执行，过程可追踪。
- 每次迭代沉淀可复用经验。
